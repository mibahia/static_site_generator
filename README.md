# Static Site Generator

A static site generator built in Python as part of a guided project from Boot.dev.

This project converts structured content into static HTML pages, simulating how tools like Jekyll work under the hood.

## Features

- Parses input files and generates HTML output
- Supports templating for consistent page structure
- Local development server for previewing pages
- Lightweight and dependency-minimal

## What I Learned

Although it's a guided project, the lessons were very difficult! While building this, I:

- Spent significant time reading error messages and tracing bugs
- Improved my understanding of file I/O, program structure and how to break down big task into smaller ones
- Learned how static site generators work internally

## Setup

This project uses `uv` for environment and dependency management.  
See the official docs: https://docs.astral.sh/uv/

### Run locally

Run:
   ```
   bash ./main.sh
   ```
Open browser at:

```
http://localhost:8899
```
